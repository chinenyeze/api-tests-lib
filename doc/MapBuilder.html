MapBuilder

Package: com.perkbox.util

Instance Methods

public MapBuilder()

The Constructor: which creates a HashMap of Map<String, Object> type. 
        MapBuilder map = new MapBuilder();

public Map<String, Object> getMap()

This method is used to get the Map<String, Object> object type from the MapBuilder. This is useful for using MapBuilder in a context where HashMap or Map is required. 
        MapBuilder mapBuilder = new MapBuilder();
        ...
        Map<String, Object> map = mapBuilder.getMap();

public MapBuilder add(String key, Object value)

This method is used to add Object values to Map by a String key. The benefit of using MapBuilder over Java's HashMap directly is that the MapBuilder's add(key, value) returns a MapBuilder object which allows for nesting, thus making your code tidy, as opposed to HashMap's put(key, value) which returns void. E.g.: 
        MapBuilder map = (new MapBuilder())
                .add("key1", "value1")
                .add("key2", "value2")
                .add("key3", "value3");
Instead of the traditional Java way (which does not support nesting): 
        Map<String, Object> map = new HashMap<>();
        map.put("key1", "value1");
        map.put("key2", "value2")
        map.put("key3", "value3");
