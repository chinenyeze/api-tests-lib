Text

Package: com.perkbox.util

Static Members

public static final String ALPHA

This is a text helper that returns all alphabets, all in lowercase. Can be useful in asserting that a feature allows lowercase alphabets.

public static final String CAPS

This is a text helper that returns all alphabets, all in uppercase. Can be useful in asserting that a feature allows uppercase alphabets.

public static final String NUMBERS

This is a text helper that returns all numbers. Can be useful in asserting that a feature allows numbers.

public static final String SPECIAL

This is a text helper that returns special characters <([{\^-=$!|]})?*+.>&. Can be useful in asserting that a feature allows special characters.

public static final String ALL

This is a text helper that returns a concatenation of the above ALPHA, CAPS, SPECIAL and NUMBERS.
